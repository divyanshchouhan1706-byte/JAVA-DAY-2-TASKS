# JavaBank - Console Banking Application

A menu-driven Java console application built as part of an internship assignment.

## Features
- Create Savings / Checking accounts
- Deposit & Withdraw funds
- Transfer funds between accounts
- View account statement & transaction history
- Calculate simple interest

## Project Structure
- `Main.java` - Entry point & menu navigation
- `Bank.java` - Manages all accounts
- `Account.java` - Core banking logic
- `Customer.java` - Customer details
- `Transaction.java` - Transaction logging

## How to Run
```bash
javac Main.java Bank.java Account.java Customer.java Transaction.java
java Main
```

## Concepts Used
- Object-Oriented Programming (OOP)
- Parameterized Constructors
- ArrayList & Collections
- Input validation with Scanner
